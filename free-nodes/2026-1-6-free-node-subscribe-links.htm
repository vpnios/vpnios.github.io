一、什么是Quantumult？——不仅是一个代理工具

Quantumult 是一款为 iOS 平台量身打造的网络代理工具，由中国开发者推出，尽管界面看起来极简，但背后功能却极为强大。

它不仅支持包括 HTTP、HTTPS、SOCKS5、VMess、Trojan、Shadowsocks 等在内的多种代理协议，还内置了 图形化规则编辑器、分流系统、网络监控与重定向功能。它可以让你精确控制每一个网络请求的走向，并对节点连接状况一目了然。

核心优势总结：

  • 支持协议广泛：主流协议几乎全覆盖；

  • 高可定制性：分流规则、策略组灵活自由；

  • 数据可视化：实时查看流量与请求路径；

  • 轻巧高效：在iOS平台上几乎无资源压力。

二、准备工作：下载安装Quantumult

1. 从App Store获取应用

  • 打开 App Store；

三、获取可用服务器节点信息

在使用Quantumult连接网络之前，用户必须获取一个合法、可用且与协议匹配的代理服务器信息。

常见获取方式：

  • 购买代理服务：选择信誉良好的服务商（如V2ray、Trojan、Shadowsocks提供商）；

  • 订阅链接：通过提供商获取订阅URL，一键导入；

  • 扫描二维码：服务商通常提供配置二维码，便于扫码添加；

  • 自建服务端：高阶用户可部署自有服务器（例如搭建VMess或Trojan节点）。

一个典型的VMess配置内容包含：

  • 地址（IP或域名）

  • 端口号

  • UUID（用户识别码）

  • 加密方式

  • TLS状态（是否启用）

  • 传输协议（如TCP/ws）

  • 路径与主机名（ws使用时必填）

四、在Quantumult中添加服务器节点的详细步骤

一旦你获得了服务器信息，接下来就是将这些信息导入Quantumult。我们分别介绍手动添加与订阅导入两种方式。

✅ 方法一：订阅链接导入（推荐方式）

  1. 打开Quantumult；

  2. 点击右下角“设置”图标，选择【配置文件】；

  3. 点击右上角“+”图标；

  4. 选择“订阅”；

  5. 粘贴订阅链接，点击“完成”；

  6. 返回主界面，手动点击“更新”图标，拉取最新节点。

这样，一批节点会自动导入Quantumult，方便你一键切换、自动测速、策略分组。

✅ 方法二：手动添加服务器

如果你只有单独的节点信息或二维码：

  1. 打开Quantumult主界面；

  2. 点击右上角“+”图标，选择“手动添加”；

  3. 根据服务器提供的协议类型选择：VMess、Trojan、Shadowsocks等；

  4. 按要求填写地址、端口、UUID、加密方式、TLS状态等信息；

  5. 为节点命名，如“东京CN2高速”或“美西GIA专线”；

  6. 点击保存，节点即添加完成。

如果是二维码，则可在“设置” → “配置”页面点击二维码图标进行扫码添加。

五、配置策略组与分流规则

Quantumult最强大之处在于它的规则机制，可以决定哪些流量需要走代理、哪些直连、哪些拦截。

1. 创建策略组

策略组可以将多个节点集合在一起，根据不同用途进行分配，例如：

  • 【全球加速】：访问海外站点

  • 【国内直连】：保留访问国内App（如微信、淘宝）

  • 【广告拦截】：自动阻断追踪链接

进入“设置” → “策略组”可添加或编辑这些组。

2. 添加分流规则

  • 点击“设置” → “规则”；

  • 你可以选择导入通用规则（如GFWList、ChinaList）；

  • 或者手动新增：

    • DOMAIN-SUFFIX,google.com,全球加速

    • DOMAIN-SUFFIX,baidu.com,国内直连

    • IP-CIDR,127.0.0.0/8,DIRECT

支持的匹配方式包括：域名后缀、IP段、端口范围、关键字等。

六、连接服务器与状态检测

  1. 返回Quantumult主界面；

  2. 选择你想启用的服务器节点；

  3. 点击右上角“开关”图标，启用VPN代理；

  4. 状态栏出现VPN图标，表示连接成功；

  5. 此时访问被屏蔽的网站或App（如Telegram、Google）应能正常加载。

Quantumult还支持 延迟测速，在“服务器列表”长按节点选择“Ping测试”，可测出响应时间。

七、实用技巧与优化建议

✅ 自动测速并切换节点

  • 设置策略组类型为“延迟最低自动选择”；

  • Quantumult将自动在多个节点中选择延迟最低的那个；

  • 特别适合网络不稳定时提升连接体验。

✅ 设置DNS防污染

  • 在“设置” → “DNS”中自定义DNS服务器：

    • 推荐：1.1.1.1, 8.8.8.8, 223.5.5.5

  • 开启“DoH/DoT”选项提高解析安全性；

  • 可启用“Fake IP”加快解析速度，减少DNS污染。

✅ 自动更新订阅配置

  • 在订阅链接下拉时自动刷新；

  • 可开启定时自动刷新功能，确保节点信息始终有效。

八、常见问题解答（FAQ）

Q1：Quantumult无法连接服务器怎么办？

排查方式如下：

  • 检查节点地址、端口、UUID等是否填写正确；

  • 检查网络状态（建议切换蜂窝/Wi-Fi尝试）；

  • 更换节点看是否是服务器问题；

  • 确保Quantumult已获得VPN权限。

Q2：如何导入多个订阅？

进入“设置” → “配置文件” → “+”，可以添加多个订阅链接，系统将自动合并节点信息，并归类至不同策略组。

Q3：Quantumult支持哪些协议？

支持范围极广，包括但不限于：

  • Shadowsocks

  • VMess（v2ray）

  • Trojan

  • SOCKS5

  • HTTP/HTTPS

同时也支持自定义混合使用不同协议节点，非常灵活。

Q4：如何拦截广告？

Quantumult可通过规则拦截广告：

  • 导入广告拦截规则（如AdGuard、EasyList CN）；

  • 使用策略组设置为“拒绝连接”；

  • 配合Safari内容拦截器，双重屏蔽效果更佳。

FAQ

PassWall2 的延迟测速功能如何提高网络体验？
PassWall2 可以定期测速节点延迟，并结合策略组自动选择最优节点。这样在访问网站或视频时，可以始终使用延迟最低的线路，提高网络稳定性和速度。
Surfboard 的策略组可以根据延迟自动切换节点吗？
支持。Surfboard 可定期测速节点延迟，并结合策略组自动选择延迟最低的节点。这样用户无需手动切换节点即可保持高速访问，提高网络体验和稳定性。
如何在 Nginx 上为 Vmess/WS 配置反向代理与 TLS？
在 Nginx 中配置 HTTPS 站点并将特定路径（如 /ray）反向代理到 V2Ray 的 WebSocket 端口，确保 proxy_set_header 设置正确（Host、X-Real-IP 等）。同时使用 certbot 获取并自动续期 TLS 证书以维护安全。
PassWall2 是什么软件？
PassWall2 是一个运行在 OpenWRT 路由器上的插件，用于代理流量分流和节点管理。它支持多种协议，包括 V2Ray、Trojan、SSR 等。通过 Web 界面，用户可以灵活设置不同设备走不同代理，适合家庭或小型局域网使用。
Nekoray 的订阅更新支持自动导入节点吗？
支持。Nekoray 可以通过订阅 URL 自动下载节点，并根据用户配置更新本地节点列表。定期更新确保节点始终可用，减少手动维护的麻烦，提高使用便利性。
Clash 的 Rule Provider 有什么作用？
Rule Provider 是 Clash 的规则分发机制。用户可以订阅不同的规则集，比如广告拦截、社交媒体、流媒体等，实现按需加载。它让配置文件更加简洁、动态更新，并支持多源规则管理。
Nekoray 节点订阅更新失败怎么办？
首先检查订阅 URL 是否有效，再确认网络是否通畅。必要时可手动刷新或重新添加订阅链接，保证节点列表最新。结合策略组可保证稳定连接和科学上网体验。
如何安全地共享订阅链接而不泄露个人信息？
避免将私人订阅直接在公开场合分享，若需要共享可在服务端生成只读或限速专用订阅，并定期更换 token。对外公开时移除或隐藏用户识别字段，确保他人无法通过订阅追溯出个人账户信息。
如何识别节点是否被封锁或限速？
通过对比不同节点的 speedtest、ping、下载测速以及 traceroute 路由变化来判断；若多个节点同时异常可能是本地网络问题，单个节点异常则可能被封锁或运营商限速，应尝试更换端口或传输方式。
Hiddify Next 如何设置用户带宽限制？
管理员可为每个用户设置独立带宽和流量配额，超出时可自动降速或阻止访问。这样可确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络效率和公平性。
